# Profile Analyzer

You’re not dating. You’re branding.

## Overview

Profile Analyzer is a web tool that examines how your online dating persona is perceived. It highlights the gap between authentic identity and curated presentation shaped by social media.

## What It Does

- Identifies your projected persona  
- Reveals unintended signals in your behavior  
- Describes how others likely perceive you  
- Compares authenticity vs performance  
- Reflects how you might come across without social media influence  

## Concept

Modern dating has become optimized and performative.  
This project explores how social media encourages strategy over genuine connection.

## Tech Stack

- HTML, CSS, JavaScript  
- Optional LLM API for dynamic analysis
